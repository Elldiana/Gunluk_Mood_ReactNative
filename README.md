# Günlük Mood 📱💭

Bu proje, kullanıcıların günlük duygu durumlarını ve dijital alışkanlıklarını takip ederek anlamlı verilere dönüştürmelerini sağlayan mobil bir uygulamadır. İzmir Ekonomi Üniversitesi Mobil Programlama dersi final projesi kapsamında geliştirilmiştir.

## 🚀 Kullanılan Teknolojiler

* **Frontend:** React Native, Expo
* **Backend & Veritabanı:** Convex (Gerçek zamanlı veri senkronizasyonu)
* **Paket Yöneticisi:** npm
